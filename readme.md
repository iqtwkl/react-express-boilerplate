# Masiflog Dashboarding 

## Tech Stack

- React (frontend)
- Vite (frontend)
- Express (Backend)
- Typeorm
- Typescript

## How to install

### install frontend

```
cd frontend
yarn install 
```

### install backend

```
# at root project
yarn install
cp env.example .env
# change your environment variables
```

## How to run

### run frontend

```
cd frontend
yarn run start 
```

### run backend

```
# at root project
yarn run start
```

## Setting Database

setting database at file
```
app/src/configs/db.config.ts
```

### initial seed
```
yarn run db:seed
```
